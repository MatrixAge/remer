# remer | based vw = 375

### How to use

add this to html head:

`<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">`
 
### used in mobile

pt => rem | (x)pt = (x/16)rem

### used in miniapp

rpx => rem | (x)rpx = (x/32)rem

rem => rpx | (x)rem = (32)rpx
